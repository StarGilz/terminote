# TermiNote v1.2

A terminal-style encrypted memo pad built with Python & Tkinter.

## Key Features
- **File Browser**: Open files using arrow keys (`:ls`).
- **Grep Search**: Find keywords across all notes (`:grep keyword`).
- **Encryption**: Notes are stored in Base64 format.
- **Vim-like Commands**: Use `:w`, `:q`, `:h` for control.

## How to Run
```bash
python main.py
